Items in archives folder are used for testing purpose only and it is not directly related to the project

1. To start the game, user needs to initiate nodeJS server environment and setup a database for the game to connect

2. The pm_game.sql file contains the database, and user need to modify the in-game database configuration located in server directory (db.js)

3. To initiate the server, user need to redirect to the server directory in a terminal/command prompt, then key in "node app.js"

4. Then, the user can play the game through a browser by keying in the URL "localhost:3000"
